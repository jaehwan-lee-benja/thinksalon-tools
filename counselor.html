시작하기 — Claude API 키 연결

이 상담사는 Claude(Anthropic)로 구동됩니다. 당신의 API 키를 넣으면 바로 실제 상담을 시작합니다. 키는 이 브라우저에만 저장되고, 대화는 Anthropic 외 어디로도 전송되지 않습니다(서버 없음).

키가 없으면 console.anthropic.com에서 발급. 사용량만큼 당신 계정에 과금됩니다.
주의 — 브라우저에서 키를 직접 쓰는 방식(BYO-key)은 개인 사용/체험용입니다. 여러 사용자에게 공개 서비스할 땐 키를 서버(Edge Function)에 숨기는 백엔드가 필요합니다(설계 별도). · 이 상담은 전문 의료·법률·재무 조언이 아니며, 위기 상황엔 전문기관 도움을 받으세요.
